## Quick Reference

**Product:** Luxury skincare brand landing page with editorial "Botanical Atelier" design aesthetic.

**Stack:** React 19 + TypeScript + Tailwind CSS 4 + Vite 7 + Framer Motion + shadcn/ui.

**Design:** Warm earthy palette (cream, olive, gold, espresso), Cormorant Garamond + Jost typography, asymmetric editorial layouts.

**Status:** Landing page complete with 14+ sections, dark mode, responsive design, and SEO optimization. Documentation suite in progress.
